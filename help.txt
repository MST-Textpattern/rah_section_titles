h1. rah_section_titles

p. Rah_section_titles makes Write tab's _section_ drop-down list show section titles instead of section names. The rah_section_titles plugin is somesort of answer to this "feature idea by Redbot":http://forum.textpattern.com/viewtopic.php?id=27744

* Version: 0.2
* Updated: 2010/06/09 05:15 AM UTC

h3. Table of Contents

* "List of features":#list-of-features
* "Requirements":#requirements
* "Installation and usage":#installation-and-usage
* "Changelog":#changelog

h3(#list-of-features). List of features

* Simple setup: just run the automatic plugin setup and activate.
* Shows section titles instead of names.
* Outputs valid XHTML, and tiny and clean JavaScript code.

h3(#requirements). Requirements

p. Rah_section_title's minimum requirements:

* Textpattern 4.0.7+
* jQuery (jQuery is bundled with Textpattern).

h3(#installation-and-usage). Installation and usage

p. The general simple behavior stands: paste the plugin code to the plugin installer box and run the automatic setup. Then just activate the plugin and it's ready, section titles are shown.
